# OBSIDIAN RELAY – Terminal Trading Post

Eine datenbankbasierte Webanwendung für ein MMORPG-Handelssystem.

## Features

- Benutzer Registrierung
- Login System
- Inventarverwaltung
- NPC Händler
- Kaufen und Verkaufen von Items

## Technologien

- Node.js
- Express
- MariaDB
- HTML / CSS / JavaScript

## Installation

1. Repository klonen

git clone https://github.com/username/obsidian-relay

2. Backend starten

npm install
node server.js
